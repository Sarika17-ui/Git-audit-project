# Git Audit Project | VIT

## Project Overview
This project audits Git and open-source practices. It includes 5 scripts, screenshots, and a report.

## Scripts
| Script Name | Purpose |
|-------------|---------|
| Script1_SystemIdentity.sh | Displays system identity information |
| Script2_PackageInspector.sh | Lists installed packages and checks for Git |
| Script3_DiskAuditor.sh | Shows disk usage and top directories |
| Script4_LogAnalyzer.sh | Analyzes recent system logs |
| Script5_ManifestoGenerator.sh | Generates a small project manifesto |
